# Cricket-Highlight-Generator-Cpp


## Overview
This project automatically generates a 10-minute cricket highlight video using multi-modal event detection.

## Features
- Audio peak detection (crowd excitement)
- Motion detection (OpenCV)
- Face detection (crowd reaction)
- Scoreboard change detection
- Automatic clip extraction and merging

## Tech Stack
- C++
- OpenCV
- FFmpeg

## How It Works
1. Input full match video
2. Detect peak moments using multiple signals
3. Score and rank events
4. Extract clips
5. Merge into highlight video

## Future Improvements
- Real-time highlights
- Deep learning integration
- Player-specific highlights
